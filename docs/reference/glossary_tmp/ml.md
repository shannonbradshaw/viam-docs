ML stands for machine learning, a field of artificial intelligence that focuses on building systems that can learn from and make decisions based on data.

Viam provides tools for [training ML models](/train/train-a-model/), [deploying them to machines](/vision/configure/), [running inference](/vision/configure/), and [interpreting visual data from cameras](/vision/alert-on-detections/) to enable intelligent behavior in robotic systems.
